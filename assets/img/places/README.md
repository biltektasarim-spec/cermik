# Çermik Rehberi - Mekan Arka Plan Resimleri Rehberi

Bu klasör (`assets/img/places/`), "Tarihi Mekanlar" ve "Kadim Rotalar" menülerindeki kartların arka plan resimlerini içerir.

## Teknik Özellikler
- **Format**: `.jpg` (Tercihen) veya `.png`
- **Önerilen Ölçü**: 800x400 piksel (veya 2:1 oranında herhangi bir yatay görsel)
- **Stil**: Görsellerin üzerine otomatik olarak koyu renk katmanı (overlay) eklendiği için çok açık renkli resimler bile okunabilirliği bozmayacaktır.
